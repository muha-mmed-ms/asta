#### 4. Get the Embed Code

1. Click the **Share** button (next to the map title)
2. In the popup:
   - Change from "Private" to **"Anyone with the link"**
   - Click **Done**
3. Click the **3 dots (‚ãÆ) menu** next to Preview
4. Select **"Embed on my website"**
5. You'll see HTML code like:
```html

```

6. **Copy ONLY the URL** (the part inside `src="..."`)
   - Example: `https://www.google.com/maps/d/embed?mid=1AbC123...`

#### 5. Update Your Website

1. Open `src/components/LocationMap.jsx`
2. Find line 9 where it says `src="/images/map.png"`
3. Replace the entire `<img>` tag with:

```jsx
<iframe
  src="YOUR_COPIED_URL_HERE"
  width="100%"
  height="100%"
  style={{ border: 0 }}
  allowFullScreen=""
  loading="lazy"
  referrerPolicy="no-referrer-when-downgrade"
  title="Asta Properties Locations"
/>
```

4. Save the file

## Why Not Use Regular Google Maps Embed?

Regular Google Maps embed URLs have complex 'pb' parameters that can only be generated by Google's interface. That's why manually creating them causes the "Invalid pb parameter" error.

**Google My Maps** is specifically designed for embedding custom maps with multiple locations, and it generates the correct URL automatically.

## Need Help?

If you have any issues:
1. Make sure the map is set to "Anyone with the link" or "Public"
2. Copy the complete URL including `?mid=...`
3. The URL should start with `https://www.google.com/maps/d/embed?mid=`

The map will update automatically if you make changes in Google My Maps later!
